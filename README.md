# university-system

This project was built to manage students' management system of university. It is possible to create new students, groups, subjects, journals, faculties, universities and more things.

# Used Technologies

✅  Java

✅  Spring-Boot

✅  Spring JPA

✅  Spring Web

✅  PostgreSql

✅  Lombok

✅  Validation

✅  Swagger

# Requirement
  JDK 11
  
  Maven 3
